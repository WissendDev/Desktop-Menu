<a name="english"></a>
## English

### Overview
High-performance launcher designed for streamers. Utilizes the native Windows Mica effect.

### Features
* **Mica Translucency:** Native Windows 11 blur effect integration.
* **Instant Management:** Remove entries directly via Right-Click.
* **Custom UI:** Smooth custom scrollbar implementation.
* **Smart Assets:** Automatic lookup and assignment of `icon.ico` files.

### Build
pyinstaller --onefile --noconsole --icon="icon.ico" --name="desktop-menu-x86-x64" main.py

<a name="russian"></a>
## Русская версия

### Описание
Высокопроизводительный лаунчер для стримеров с использованием нативного эффекта Mica.

### Особенности
* **Прозрачность Mica:** Нативная поддержка эффекта размытия Windows 11.

* **Быстрое управление:** Удаление элементов через контекстное меню (ПКМ).

* **Кастомный UI:** Реализация плавного пользовательского скроллбара.

* **Умные иконки:** Автоматический поиск и привязка файлов `icon.ico`.

### Сборка
pyinstaller --onefile --noconsole --icon="icon.ico" --name="desktop_menu-x86-x64" main.py
